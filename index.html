SF the list: iCalendar edition

This is an iCalendar version of Steve Koepke's excellent (San Francisco) Bay Area concert guide http://www.calweb.com/~skoepke/. All of the content is Steve's. Please send Steve mail if you have questions or corrections related to the list content.

This project converts the plaintext list to an iCalendar file fomat.

Plain text version of the list : http://jon.luini.com/thelist/thelist.txt Archives going back to 1994 : http://jon.luini.com/thelist/archive/
